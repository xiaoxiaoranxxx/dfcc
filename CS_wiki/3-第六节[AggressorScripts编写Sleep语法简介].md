### 运算

```
$x = 5 + "1";
$x = 5 - $y;
$x = $x * $2;
$x = $z / 9.9;

```

### 数组

定义一个数组用 `@`

```
@foo = @("a", "b", "c");                                          
$x = @foo[1];                                                     
println($x);

```

### Hash Scalars

类似字典

```
%bar = %(x => "x-ray", y => "yabboes");

```


### 函数

定义函数 `sub function` 参数为`$1,$2,$3`

```
sub add
{
    $a = $1;
    println($a);
}

```

### if-else 条件判断

```
$nub = 1;
$nub2 = 2;
if ($nub>$nub2)
{
    println("1>2");
}
else
{
    println("1<2");
}
```

### 逻辑运算符

`and(&&)` `or(||)`

```
sub check
{
if (($1 > 0) && ($2 <= 10))
{
println("$1 is within 0 .. 10");
}
else
{
println("$1 $2 is out of range!!");
}
}

```

### 循环语句

For 循环

```
for ($total = 99; $total > 0; $total--)
{
    println($total);
}

```

while 循环

```
$total = 99;
while ($total > 0)
{
    println($total);
}

```

Foreach 循环

```
%data = %(foo => "a", bar => "b", baz => "c", jaz => "d");
foreach $key => $value (%data)
{
    println($key);

}

```
